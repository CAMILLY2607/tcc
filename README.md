## Resumo

O projeto **Conecta** é uma iniciativa que reúne empresas do setor público, ONGs e instituições educacionais, como o IFPR, com o objetivo de promover a inovação. Iniciando suas atividades em dezembro de 2021, conta com mais de 100 colaboradores ativos e busca fomentar a inovação por meio de capacitações, parcerias, apoio na busca de financiamento e eventos. Em 2022, destacou-se no Prêmio Nacional de Inovação do Sebrae como um dos seis finalistas na categoria de Ecossistemas Emergentes.

![Imagem do Conecta](URL_DA_IMAGEM)

## Objetivos

Desenvolver um portal web para divulgação de informações sobre o ecossistema de inovação, permitindo que as empresas, atores ou até mesmo a sociedade tenham acesso às informações e atividades realizadas pelo grupo, promovendo a inovação, facilitando o compartilhamento de conhecimento e fortalecendo o ecossistema local.

### Objetivos Específicos

- Levantar as necessidades do grupo Conecta para estabelecer os requisitos do sistema.
- Desenvolver páginas com conteúdo dinâmico e de edição amigável para o administrador do site.
- Criar um banco de dados para cadastro das empresas pertencentes ao Conecta.
- Implementar uma área restrita onde os membros possam buscar e oferecer parcerias e colaborações para projetos inovadores.

## Justificativa

A criação do portal é justificada pela necessidade de um espaço colaborativo, onde a plataforma atuará como um ambiente virtual para a troca de ideias e formação de parcerias. Isso permitirá que todos os interessados tenham acesso facilitado a informações essenciais, promovendo uma inclusão e participação ampliadas da comunidade local.
